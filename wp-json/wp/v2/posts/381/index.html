  • A delegate is a type that represents references to methods with a particular parameter list and return type. When you instantiate a delegate, you can associate its instance with any method with a compatible signature and return type. You can invoke or call the method through the delegate instance.<\/li>\n\n\n\n
  • Delegates are used to pass methods as arguments to other methods. Event handlers are nothing more than methods that are invoked through delegates. <\/li>\n\n\n\n
  • Delegates can be used to define callback methods.<\/li>\n\n\n\n
  • Delegates can be chained together; multiple methods can be called on a single event.<\/li>\n<\/ul>\n\n\n\n
